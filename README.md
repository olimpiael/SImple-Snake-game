# Snake Game 🐍

This project is a simple implementation of the classic **Snake game**, inspired by the nostalgic version many people remember from old Nokia phones.

The game allows the player to control a pixel snake, collect food, and grow longer while trying to avoid hitting its own body. The goal is to achieve the highest possible score.

To make the gameplay more fun, the player can **choose the color of the snake before starting the game**, adding a small customization element.

The game also includes a **scoreboard system**, where players can save their results and compete with others for the best score.

## Features

- Classic Snake gameplay
- Snake color selection
- Scoreboard with saved results
- Simple graphical interface using **Tkinter**
- Keyboard controls

## Technologies

- Python
- Tkinter (GUI)

## Controls

- **Arrow keys** – move the snake  
- **P** – pause the game  
- **Escape** – exit confirmation
